# HueConnect

Simple Sinatra endpoint for controlling Phillips Hue lights

## Installation

Add this line to your application's Gemfile:

    gem 'hue_connect'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hue_connect
